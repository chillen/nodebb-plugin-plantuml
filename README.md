# NodeBB PlantUML Plugin #

This is a plugin which adds [PlantUML](http://plantuml.sourceforge.net/) support
to your NodeBB forum!

**Sample Usage**

```
Here's some sample text!

@startuml
Foo->Bar : Rawr
Bar->Foo : Sauce
@enduml

Isn't that a good diagram?
```
